### Sales Performance Dashboard - Power BI

## 📊 Project Overview

This project presents an interactive **Sales Performance Dashboard** developed using **Power BI**. The dashboard helps analyze sales data and provides valuable insights into revenue, profit, customer trends, product performance, and regional sales distribution.

## 🎯 Purpose

The main objective of this project is to transform raw sales data into meaningful visualizations that support data-driven business decisions and improve overall sales performance.

## 🚀 Features

* Total Sales, Profit, and Quantity Analysis
* Sales Trends Over Time
* Region-wise Sales Performance
* Product Category Analysis
* Customer Insights
* Interactive Filters and Slicers
* Dynamic Visualizations and KPI Cards

## 🛠️ Tools & Technologies

* Power BI Desktop
* Data Cleaning & Transformation (Power Query)
* Data Modeling
* DAX (Data Analysis Expressions)

## 📈 Key Insights

* Identifies top-performing products and regions.
* Tracks sales growth and profitability trends.
* Highlights business opportunities and improvement areas.
* Enables quick decision-making through interactive reports.
